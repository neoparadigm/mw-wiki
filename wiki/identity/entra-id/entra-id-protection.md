With Azure AD Identity Protection it is possible to protect users based on the Microsoft signals.  Azure AD Identity protection is all about risk, detection, and remediation based on the identity level. Microsoft uses threat intelligence to specify risky detection for all users.  This blog is about all the parts of Azure AD Identity Protection.**
