# Win32 App Deployment and Packaging

## Win32 App Deployment and Packaging

Win32 app deployment and packaging is a crucial aspect of managing modern workplaces, enabling administrators to distribute custom applications on Windows devices. This topic matters due to the need for organizations to deploy and manage their own applications alongside Microsoft-provided apps.

## Key Concepts
- Creating Win32 app packages using tools like Advanced Installer, WiX Toolset, or Setup Project
- Using IntuneWinAppUtil to create .intunewin files from existing MSI or EXE files
- Configuring app deployment policies in Microsoft Intune for Win32 apps
- Understanding the differences between Win32 app types: Win32 traditional, Win32 signed, and Win32 notarized
